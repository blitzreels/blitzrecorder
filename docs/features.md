# Features

## Recording

- Capture the Mac screen.
- Capture a selected display, window, or app through the screen picker.
- Record a local Mac camera, external camera, Continuity Camera, or remote iPhone camera.
- Record microphone audio.
- Record Mac system audio.
- Pause and resume while keeping the exported timeline clean.

## Layout

- Compose vertical 9:16 creator videos.
- Compose horizontal 16:9 videos.
- Position screen and camera layers on a live canvas.
- Use picture-in-picture and stacked layouts.
- Show cursor and framing overlays when useful.

## iPhone camera

- Pair an iPhone over the local network.
- Show a live monitor preview on the Mac.
- Control supported iPhone camera settings from the Mac.
- Record the iPhone camera file locally on the phone.
- Transfer the completed camera file back to the Mac take.

## After recording

- Export the finished video.
- Open the final recording.
- Reveal the export in Finder.
- Rename or move the export.
- Reveal saved source files.
- Retry export from recovery files when possible.
- Use transcript-assisted file naming.
