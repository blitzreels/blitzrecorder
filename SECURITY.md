# Security

Please do not report security issues in public GitHub issues.

Email sensitive reports to `support@blitzreels.com`.

Useful details to include:

- App version and build number.
- macOS or iOS version.
- Whether the issue affects the Mac app, iPhone companion, website, or release pipeline.
- Steps to reproduce.
- Any logs, screenshots, or sample files needed to understand the issue, with secrets removed.

BlitzRecorder is in active development, so response times may vary. I will confirm receipt when possible and follow up when there is a fix or a clear mitigation.
